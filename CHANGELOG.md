# Changelog

jstack is a fork of [gstack](https://github.com/garrytan/gstack) v0.15.1.0.

## [0.1.0] — Unreleased

Privacy-first AI engineering toolkit with Cursor IDE support.

### What's new
- **Zero telemetry.** Nothing leaves your machine. No Supabase, no installation ID, no phone-home.
- **Cursor IDE support.** MCP server for the browse daemon + .mdc rules for workflow skills.
- **jstack branding.** Clean fork identity, ready for company-specific forks via `jstack init --company`.
- **Slimmer preamble.** Faster skill startup, fewer external commands on every invocation.
- **/brainstorm** replaces /office-hours — same great forcing questions, no YC framing.

### For contributors
- Compiled binaries removed from git tracking
- Fresh CHANGELOG starting at v0.1.0
- UPSTREAM-RENAME-MAP.md documents all renamed paths for cherry-picking upstream fixes
